
# Day 4: Printing Department - Part 1

This file will document the solution approach, challenges, and explanations
for part 1.

## Approach

- Parse the input grid.
- For each roll of paper ('@'), count the number of adjacent '@' in the 8
  surrounding cells.
- If fewer than 4, count it as accessible.
